## Post Text

5 Tableau design rules that make stakeholders actually trust your dashboards:

1. One question per view
   Every sheet should answer ONE thing.
   "What is our monthly revenue trend?" — not 6 things at once.

2. Remove every default
   Default grey background. Default blue bars. Default title font.
   If Tableau chose it — question it.
   Custom colors build brand trust.

3. Use tooltips as a second layer
   Don't crowd the viz.
   Put supporting context in the tooltip.
   Hover = deeper dive. Default view = clean.

4. Align EVERYTHING
   Misaligned titles, filters, and legends destroy professionalism.
   Use Layout pane. Use containers. Be pixel-precise.

5. Always add a "Last Updated" text
   One of the most common stakeholder questions:
   "Is this data current?"
   Answer it before they ask.
   Add a text box: "Data refreshed daily at 6 AM"

The difference between a dashboard people use and one they ignore:
It's almost never the data.
It's almost always the design.

Which rule are you implementing today? 👇
